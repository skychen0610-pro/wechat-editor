============================================
     公众号排版器 - 本地部署版
============================================

【使用方法】

方式一：直接双击打开（简单模式）
   直接双击 index.html 即可在浏览器中打开使用
   注意：部分功能（如图片上传）可能受限

方式二：启动本地服务器（推荐）
   1. 双击运行 "启动.bat"
   2. 等待显示 "Serving HTTP on :: port 8080"
   3. 浏览器会自动打开，或手动访问 http://localhost:8080
   4. 按 Ctrl+C 关闭服务器

【文件说明】
   index.html  - 主页面
   styles.css  - 样式文件（19个主题）
   app.js      - 功能逻辑
   启动.bat    - 一键启动本地服务器

【系统要求】
   - Windows 7/10/11
   - Python 3.x（一般系统已预装）
   - Chrome / Edge / Firefox 浏览器

【在其他电脑上使用】
   将整个 "公众号排版器_本地版" 文件夹复制到目标电脑
   然后按上述方法使用即可
